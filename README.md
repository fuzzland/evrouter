# Ethervista Router Wrapper

Uses [Ethervista bug](https://x.com/shoucccc/status/1831707216711446842) to bypass buy fee. 


### Run Tests

```
forge test
```

### Run Frontend

```
cd frontend
pnpm i
pnpm run dev
```


### Deploy Contract

```
forge create test/Counter.t.sol:Ampl --rpc-url [RPC] --private-key [PRIVATE_KEY]
```
